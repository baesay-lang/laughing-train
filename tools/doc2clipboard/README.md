# Doc2Clipboard / Doc2Batch

파일 업로드가 막힌 PC에서, 폴더 안의 문서를 **붙여넣기(Ctrl+V)** 로 `claude.ai`에
넘길 수 있게 해 주는 도구입니다. 설치 프로그램(.exe)이 없어 AppLocker / WDAC 정책과 무관합니다.
(Windows 기본 PowerShell + 이미 깔린 MS Office만 사용)

두 개의 도구가 있습니다:

| 도구 | 하는 일 | 언제 |
| --- | --- | --- |
| **Doc2Batch.ps1** (추천) | 문서를 **PDF/이미지로 변환**해 여러 개를 **배치로 붙여넣기** | 도표·스캔·표가 있는 일반 문서 대부분 |
| Doc2Clipboard.ps1 | 문서를 **텍스트(.md)** 로 뽑아 청크로 붙여넣기 | 순수 텍스트 문서 (가볍고 정확) |

---

## ⭐ 설치 (딱 한 번) → 이후 우클릭

1. GitHub에서 [`Setup.ps1`](./Setup.ps1) → **`Raw`** → `Ctrl+A` → `Ctrl+C`
2. `Win` → `powershell` → **Windows PowerShell** 실행
3. `Ctrl+V` → `Enter`

설치되면 폴더에서 **우클릭 → 보내기(Send to)** 에 세 항목이 생깁니다:

- **1. Doc2Batch (추천)** — 폴더 문서를 PDF로 변환 → 20개씩 파일 묶음으로 클립보드
- **2. Doc2Batch 낱장이미지** — 각 페이지를 PNG로 렌더 → 한 장씩 클립보드
- **3. Doc2Clipboard 텍스트** — 텍스트만 뽑아 붙여넣기

바탕화면 `Doc2Batch` 아이콘도 생깁니다(더블클릭 → 폴더 경로 입력).

## 쓰는 법

### 1) Doc2Batch (추천) — PDF 배치

문서 폴더에서 **우클릭 → 보내기 → 1. Doc2Batch**.

- Office 문서는 **PDF로 변환**되고(도표·표·스캔 그대로 보존), PDF/이미지는 그대로 사용됩니다.
- **20개씩 클립보드에 "파일 묶음"으로** 올라갑니다.
- 채팅창에 **Ctrl+V 한 번 → 20개 첨부** → 스크립트 창에서 **Enter → 다음 20개**.

> Claude는 PDF를 텍스트 + 그림으로 함께 읽습니다(스캔본은 시각 인식). 그래서 이 방식이
> 텍스트 추출보다 원문을 더 잘 보존합니다.

**한 번에 여러 파일이 안 붙는다면** → 이 브라우저가 '파일 붙여넣기'를 막는 것이니
아래 낱장 이미지 모드를 쓰세요.

### 2) Doc2Batch 낱장이미지 — 한 장씩 확실히

우클릭 → 보내기 → **2. Doc2Batch 낱장이미지**.

- 각 페이지를 **PNG로 렌더**해서 **한 장씩** 클립보드(이미지)에 복사합니다.
- 채팅창에 **Ctrl+V → Enter → 다음 장**. 이미지 붙여넣기는 거의 모든 환경에서 됩니다.

### 3) 텍스트 모드

우클릭 → 보내기 → **3. Doc2Clipboard 텍스트**. 순수 텍스트 문서에 가장 가볍고 정확합니다.

## 설정 (환경변수)

우클릭 실행에도 적용됩니다. 콘솔에서 실행 전 먼저 설정하면 됩니다.

| 환경변수 | 설명 | 기본값 |
| --- | --- | --- |
| `D2C_FOLDER` | 대상 폴더 (없으면 물어봄) | (없음) |
| `D2C_BATCH` | 한 배치당 파일 수 (Doc2Batch) | 20 |
| `D2C_SINGLE` | 값 있으면 낱장 이미지 모드 | off |
| `D2C_SCALE` | PNG 렌더 배율 (클수록 선명·용량↑) | 2.0 |
| `D2C_CHUNKSIZE` | 붙여넣기 1회당 글자 수 (텍스트 모드) | 15000 |

## 알아두면 좋은 점 / 한계

- 첫 실행 때 Word/Excel/PowerPoint가 잠깐 백그라운드로 떴다 사라집니다 — 정상입니다.
- **정책이 Office COM 자체를 막는 환경**이면 자동 변환이 불가합니다. 이때는 스크립트가
  "차단됨"이라고 알려 주며, 그 경우 `Win+Shift+S` 화면 캡처 → `Ctrl+V`가 유일한 길입니다.
- 낱장 이미지 모드에서 PDF→PNG 렌더는 Windows 10/11 내장 기능(WinRT)을 씁니다.
  이게 막힌 드문 환경이면 "WinRT 렌더 불가"로 표시됩니다.
- 변환 결과물은 폴더 안 `_chat_export`(배치) / `_md_export`(텍스트)에 남습니다.
